### 2.3.1 문자열의 추출
```py
s = "안녕하세요. 반갑습니다."
```
인덱스 0
```py
s[0]
```
인덱스 3:10 (인덱스 3부터 10미만)
```py
s[3:10]
```
### 2.3.2 integer 를 String으로 바꿔주기
```py
score= 80
print("성적: "+ score)
```
이렇게 하면 문제가 생김 string + integer니까
해결방법 str() 사용
```py
print("성적: " + str(score))
```
