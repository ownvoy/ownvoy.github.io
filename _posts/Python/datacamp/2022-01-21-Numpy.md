## Numpy

### Numpy array 만들기

```python
# Create baseball, a list of lists
baseball = [[180, 78.4],
            [175, 102.7],
            [190, 98.5],
            [188, 75.2]]

# Import numpy
import numpy as np

# Create a 2D numpy array from baseball: np_baseball
np_baseball = np.array(baseball)
print(np_baseball)

```

    [[180.   78.4]
     [175.  102.7]
     [190.   98.5]
     [188.   75.2]]
    
<br>
<br>

### shape 살펴 보기
`shape` 메소드를 사용하면 된다.  


```python
# Print out the shape of np_baseball
print(np_baseball.shape)
```

    (4, 2)
